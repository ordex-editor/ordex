# Commands

Commands are entered from normal mode by pressing `:`.

## Save

```text
:w
```

Writes current buffer changes to disk.

## Quit

```text
:q
```

Exits the editor.

## Save and Quit

```text
:wq
```

Saves, then exits.

## Go to Line

```text
:{number}
```

Examples:

- `:1` goes to the first line
- `:50` goes to line 50
